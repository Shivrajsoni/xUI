## Getting started

All xUI components are built with [Tailwind CSS v4](https://tailwindcss.com/docs/installation/framework-guides/nextjs) and [React 19](https://react.dev/). Make sure Tailwind is set up in your project before adding components.

### 1. Install the `cn` utility

Every component uses the `cn` helper for class names. Add it with the shadcn CLI:

<Tabs items={["npx", "bun", "pnpm"]}>

```bash tab="npx"
npx shadcn@latest add https://x-ui-self.vercel.app/r/utils.json
```

```bash tab="bun"
bunx shadcn@latest add https://x-ui-self.vercel.app/r/utils.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://x-ui-self.vercel.app/r/utils.json
```

</Tabs>

### 2. Add a component

We use [lucide-react](https://lucide.dev/guide/installation) for icons and primitives from [shadcn/ui](https://ui.shadcn.com/). The CLI installs any required dependencies automatically. For example, to add the `card-01` component:

<Tabs items={["npx", "bun", "pnpm"]}>

```bash tab="npx"
npx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json
```

```bash tab="bun"
bunx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json
```

</Tabs>

### 3. Use it

```tsx
import Card01 from "@/components/xui/card/card-01";

export default function Page() {
  return <Card01 />;
}
```

## Shorter installs with a namespace

Tired of pasting long URLs? Register xUI once in your `components.json` and
install any component with a short `@xui/<name>` handle:

```json title="components.json"
{
  "registries": {
    "@xui": "https://x-ui-self.vercel.app/r/{name}.json"
  }
}
```

Then:

```bash
npx shadcn@latest add @xui/card-02
```

### Ways to add a component

- **Namespaced CLI** — `npx shadcn@latest add @xui/card-02` (after the one-time `components.json` setup above). Shortest.
- **Direct CLI** — `npx shadcn@latest add https://x-ui-self.vercel.app/r/card-02.json`. No setup, works anywhere.
- **Copy** — grab the source from any component's docs preview or from Studio.
- **Studio** — open a component in [Studio](/studio), customize it live, then copy or export.

<Callout>
  Prefer the CLI over copy-paste — it pulls in every required file and
  dependency for you.
</Callout>

### Using a monorepo

If you're in a monorepo, point the CLI at your app with `-c`:

<Tabs items={["npx", "bun", "pnpm"]}>

```bash tab="npx"
npx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json -c ./apps/www
```

```bash tab="bun"
bunx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json -c ./apps/www
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://x-ui-self.vercel.app/r/card-01.json -c ./apps/www
```

</Tabs>
